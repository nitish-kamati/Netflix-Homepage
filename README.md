# 🎬 Netflix Homepage Clone

A responsive **Netflix Landing Page Clone** built using **HTML, CSS, and JavaScript**.  
This project replicates the UI of Netflix's homepage including the landing page, sign-in page, and basic navigation flow.

The main goal of this project was to practice **frontend development concepts**, UI structuring, and project organization similar to real-world applications.

---

## 🚀 Features

- Netflix-style **landing page**
- **Sign In page UI**
- Fake login system using **JavaScript**
- Responsive layout
- Organized project folder structure
- Clean and modern UI inspired by Netflix

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Git & GitHub**
